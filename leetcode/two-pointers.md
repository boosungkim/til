# Two Pointers
Using two pointers to traverse arrays and linked lists efficiently.

Key concepts:
- Fixed sliding window
- Dynamic sliding window

Key patterns:
- Find the middle element
- Find the kth element from the end of the list
- Optimize problems involving pairs, such as sum or difference
- Contiguous subarray needed as window
